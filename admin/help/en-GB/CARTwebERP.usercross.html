CARTwebERP Extension

User Cross

If you use CARTwebERP, please post a rating and a review it at the Joomla! Extensions Directory .

The user cross table stores the cross reference between Joomla users and webERP customers. It is necessary to ensure the purchases by a Joomla user are always made against one webERP customer account. Records in this table are automatically created when a user that is not logged in makes a purchase. Data from the purchase is used to create a customer record in webERP and a Joomla user record. The email address is used as the user name. An email is sent with Joomla log in information on the first purchase.

Use this menu selection to add more cross references or delete them. If you have existing customers in webERP, you will need to create a Joomla user record for them. Use this program to create the Joomla to webERP cross reference record. We recommend using email addresses for the Joomla User name for consistency with automatically created accounts.

  1. Click on Components->CARTwebERP-> User-Customer
  2. To add a cross reference, click "New".
  3. This will take you to a new screen where you choose from customers and users. In order to reassign users, you just make a new association. The new one will replace the old one. Multiple users may be assigned to one customer
  4. Click the radio selection next to the user and customer to associate them.
  5. Click Save in the upper right corner.
